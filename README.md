# KamiTV

Benvenuto nella repository di rilascio di **KamiTV**!

Qui puoi trovare l'ultimo file APK compilato e le relative note di rilascio utilizzate dall'applicazione per gli aggiornamenti OTA automatici.

## Installazione
1. Scarica il file `KamiTV.apk` dall'ultima Release di questo repository.
2. Apri il file sul tuo dispositivo Android.
3. Se richiesto, abilita l'installazione da "Sorgenti Sconosciute" per il tuo browser o gestore file.
4. Completa l'installazione e avvia l'app!
